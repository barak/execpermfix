
h1. NAME

execpermfix - fixes executable permissions

h1. SYNOPSIS

**execpermfix** [-h]

**execpermfix** [-v] [-n] &lt;files&gt;

h1. DESCRIPTION

**execpermfix** is a program that fixes executable permissions.

It detects which files are executable by checking their names and contents, and gives the executable permission to everyone that has read permissions.

h1. OPTIONS

**-h** Help.

**-v** Verbose mode.

**-n** Dry-run: no operation is performed. Use with _-v_ to see files with "wrong" permissions.
